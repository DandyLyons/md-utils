# `md-utils`

Use `md-utils` to inspect and transform Markdown files from the command line.

## Overview

The executable target wraps the `MarkdownUtilities` library in a Swift Argument Parser command hierarchy. Commands operate on Markdown files or directories, with shared path behavior for recursion, hidden files, extension filtering, and sorted output.
