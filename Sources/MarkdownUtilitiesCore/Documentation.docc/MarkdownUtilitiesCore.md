# ``MarkdownUtilitiesCore``

Parse, inspect, transform, and export Markdown content from Swift code.

## Overview

`MarkdownUtilitiesCore` provides focused APIs around parsed `MarkdownDocument` values and canonical `MarkdownRecord` resources. It also provides portable Markdown type and rule assessment with structured diagnostics and fix-its.

Use this target for content-only Markdown operations that must run on Apple platforms and Linux without direct filesystem or process access.
